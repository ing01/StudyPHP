ATENÇÃO

  • NÃO é possível executar qualquer função sem preencher todos os campos requeridos.
  • Para consultar um único time: apenas o nome preenchido corretamente já realiza a consulta.
  • Listagem exibe todos os times: apenas o nome preenchido corretamente já executa.
  • Para excluir um único time: apenas o nome preenchido corretamente.
  • Para alterar um único time: não é possível alterar o nome, somente os demais dados.
